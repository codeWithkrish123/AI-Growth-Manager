## 🎯 Deploy with Blueprint (2 minutes)

### Step 1: Go to Render
https://dashboard.render.com/

### Step 2: Click "Create New" → "Blueprint"

### Step 3: Select Your GitHub Repo
- Choose: `AI-Growth-Manager`
- Confirm

### Step 4: Review & Deploy
- Render reads `render.yaml`
- Creates:
  - ✅ PostgreSQL database
  - ✅ Web service (backend)
  - ✅ Environment variables
- Click "Create Blueprint"
- Wait 5-10 minutes


## 🔗 Connect Frontend

1. Go to Vercel dashboard
2. Settings → Environment Variables
3. Add: `VITE_BACKEND_URL = https://your-backend-XXXX.onrender.com/api`
4. Redeploy frontend
